# Proposta Comercial — Arhàndu Ka'a Cosméticos Naturais

Apresentação interativa (single-file HTML) construída sobre o Design System da Glifo
(Marketing + Dev + Audiovisual).

## Conteúdo
- `index.html` — deck completo com toggle **Proposta Completa / Proposta Otimizada**
- `ref arandu *.jpg` — moodboard de referência (Etapa 02 / Fase 2)

## Como abrir
Abra `index.html` no navegador. Imprime em PDF (Cmd+P) com quebra por slide.

## Escopo
- **Completa** — Etapa 01 (branding, plataforma+ERP, audiovisual, fornecedores) + Etapa 02 (posicionamento, Instagram brandproduct, produção audiovisual, documentários, UGC).
- **Otimizada** — Fase 1 (branding essencial) + Fase 2 (comunicação, audiovisual, documentários, UGC).

Documento comercial confidencial — contém valores e condições.
